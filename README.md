# HitamLegam Top

Cyber true-black landing page for [hitamlegam.top](https://hitamlegam.top).

## Stack

- Static HTML
- Tailwind CDN
- Phosphor Icons CDN
- Google Fonts
- Cloudflare Pages deployment

## Files

| File | Purpose |
| --- | --- |
| `index.html` | Main landing page |
| `logo.jpg` | Mascot/logo asset |
| `favicon.ico` / PNG icons | Browser and mobile icons |

## Deploy

Current production target:

```txt
https://hitamlegam.top
```

Cloudflare Pages project:

```txt
hitamlegam-top
```

## Notes

This repo keeps the production landing page source public and easy to redeploy.
